## Projeto  
O projeto consiste no sistema para catálogo de produtos. No momento está sendo desenvolvido o back end com as seguintes tecnologias: 
- Java
- Spring Boot
- Banco H2 para testes locais 
- Banco PostgreSQL
- Postman para equisições feitas pela API
- Padrão REST e DTO
